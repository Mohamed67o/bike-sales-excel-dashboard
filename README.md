# Bike Sales Excel Dashboard

## Project Overview

This project analyzes customer bike purchase behavior using Microsoft Excel.

## Skills Demonstrated

- Data Cleaning
- Sorting & Filtering
- Pivot Tables
- Pivot Charts
- Dashboard Design
- Data Visualization

## Tools Used

- Microsoft Excel

## Key Questions Answered

- Which age groups buy bikes most frequently?
- Does income affect bike purchases?
- How does marital status influence purchasing behavior?
- Which regions generate the most bike sales?
